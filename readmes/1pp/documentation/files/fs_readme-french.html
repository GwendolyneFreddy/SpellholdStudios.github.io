Présentation


La vocation de ce mod est très simple : il s'agit de s'assurer que toutes les épées enflammées du jeu aient l'apparence de véritables épées enflammées !

Consultez le site du mod ou son forum pour obtenir les dernières mises à jour.

Contenu


En bref, ce mod transforme toutes les épées enflammées du jeu en véritables épées enflammées. Il remplace les animations FS (flaming sword) par des animations d'épée enflammée.


« Il s'agit d'un contenu labellisé 1ppv3, ce qui signifie qu'il est défintivement achevé (pour n'importe quelle combinaison de race/sexe). Mais ce n'est pas tout, j'ai complètement redessiné les fichiers par souci de cohérence. J'avais remarqué des erreurs plutôt embarrassantes qui ne me convenaient pas dans la version précédente. Pour l'essentiel, vous avez sans doute remarqué que la catégorie des animations M (elfes ?) semblait bien moins moins vive que la série des animations N (humaines).

Je l’aurais publié il y a quelques jours déjà, si un bug de dernière minute assez inattendu ne s'était pas manifesté à mon attention. J'ai essayé de le corriger, mais pas correctement, ce qui signifie que je ne suis pas - encore - totalement satisfaite. Vous remarquerez peut-être ce que je veux dire.

Espérons qu'il n'y aura pas de problèmes, mais n'hésitez pas à les signaler si vous en trouvez. »

- Erephine.




1ppv Épées courtes enflammées

Contenu :

  • Les animations d'épées courtes enflammées et les images des paperdolls.
  • La modification des objets, à la fois du jeu et de certains mods.
  • Correction de quelques épées enflammées « classiques » qui n'étaient pas modifiées par le code de Nythrun.

Notes :

Il s'agit d'une version mise à jour du mod original qui patche les objets et ne les écrase plus, dans un souci de compatibilité avec les autres mods.



Comment l'utiliser ?

Si vous voulez créer un mod qui utilise une épée courte enflamméee, la procésure est assez simple : créez ou modifiez une épée courte normale. Si vous souhaitez que l'épée brille dans l'obscurité (la plupart des épées enflammées le font), n'oubliez-pas d'attribuer à la lame un effet « colourglow_solid » [NdT : opcode #8] à l'emplacement 21 [NdT : la lame] dans NI.

Modifiez l'animation de l'objet en remplaçant le code de base « SS » par « F2 ».

Tout fonctionnera bien si ce composant est installé, cependant, si ce n'est pas le cas, le jeu se bloquera lorsqu'un personnage essaiera de s'équiper de votre épée. Alors, assurez-vous que les joueurs potentiels de votre mod l'auront bien installé (par exemple en le fournissant avec votre mod et en l'installant d'office s'il ne l'est pas déjà).
