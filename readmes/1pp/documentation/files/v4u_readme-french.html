Notez que si vous choisissez l'option « DISPERSER les BOUCLIERS À TRAVERS LE JEU », vous bénéficierez de boucliers utilisant à la fois le design de BG1 et de BG2, en une seule installation. Les plus fréquents seront ceux du style que vous aurez défini comme choix principal. Il en va de même pour les armures, et, dans une moindre mesure, pour les casques.

À propos des couleurs des objets

Si vous choisissez d'utiliser des couleurs pré-établies pour les objets non magiques, l'apparence des boucliers et des autres objets ordinaires ressembleront à leur icône d'inventaire lorsque vous vous en équiperez ; vous ne pourrez donc pas les « customiser » aux couleurs de vos vêtements, comme c'est déjà le cas pour les objets magiques dans le jeu vanilla (ce qui est cohérent, et, si vous me le demandez, sensé. Il est tout simplement impossible de modifier la couleur d'un bouclier ou d'un casque). Cependant, comme je suis certaine que beaucoup allaient se plaindre, cette option est seulement désactivée par défaut et vous pouvez la modifier (l'option 3 conserve ce réglage pour les boucliers non magiques au style de BG1).
